# Youtube Downloader
----------
This is a simple youtube downloader that can download videos and playlists from youtube. It is written in python and uses the yt-dlp library.
----------
Dependencies:
- [Python 3.6+](https://www.python.org/downloads/)
- yt-dlp -> `pip install yt-dlp`
- [ffmpeg](https://ffmpeg.org/download.html)
----------
Usage:
- [Download the zip file and extract it](https://github.com/Akuras88/youtube-downloader/releases/tag/release)
- Run the `youtube_downloader.exe` file
- Enter the link of the video or playlist you want to download
- Wait for the download to finish
- The downloaded files will be in the `downloads` folder which will automatically be created and opened
----------
